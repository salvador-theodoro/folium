Sobre o Projeto

Este site foi criado como um projeto de TCC por alunos do curso técnico de Informática da ETEC Jorge Street, em São Caetano do Sul (SP), com o objetivo de oferecer um espaço moderno, acessível e funcional para a publicação e organização de portfólios de projetos. A plataforma permite registrar, organizar e divulgar o desenvolvimento de projetos de qualquer tipo — sendo útil tanto para estudantes em formação, que desejam apresentar seus trabalhos desenvolvidos ao longo do curso, quanto para profissionais que buscam uma alternativa mais personalizada e independente às plataformas tradicionais.

Objetivo

A proposta é permitir que qualquer pessoa possa registrar seus projetos de forma estruturada, visual e fácil de navegar — incluindo descrições, arquivos, imagens, vídeos e links. É ideal para quem deseja apresentar seu trabalho de forma clara e impactante, seja para fins acadêmicos, profissionais ou pessoais.

Tecnologias Utilizadas

  • HTML, CSS e JavaScript para estrutura, estilo e interatividade.
  • PHP no backend para organizar os envios e postagens.

Para quem é esta plataforma?

A plataforma foi pensada para atender tanto profissionais que desejam expor seus trabalhos de forma independente quanto alunos que estão iniciando sua carreira e ainda não possuem experiência profissional. Para os estudantes, é uma forma de reunir os projetos desenvolvidos durante o curso e apresentá-los ao mercado. Para os profissionais, é uma alternativa moderna e personalizável ao GitHub e outras plataformas tradicionais.
